# component
A poi.js preset for Vue Class Component and ES6.

## Uses

* babel-preset-env: ^1.6.0,
* babel-plugin-transform-class-properties: ^6.24.1
* babel-plugin-transform-decorators-legacy: ^1.3.4

## Instructions
```js
// poi.config.js
module.exports = {
    presets: [
        require('poi-preset-component')()
    ]
}
```
